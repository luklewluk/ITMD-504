# ITMD 504 - Programming Foundations project

## Description
This is a project for ITMD 504 course.

Application is a Shopping List created from predefined products with the option to add new products.

## Stack
### Frontend
- Bootstrap
- React
- node 22

### Backend
- Symfony 8.1
- PHP 8.4

### Server and deployment
#### Backend
