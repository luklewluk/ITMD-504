# Design application

## Model
Gemini 3.1 Pro Preview in Google AI Studio

## Prompt
> Create a basic website template for shopping list that only shows list of already added items and there should be option to edit them or remove + add button that allows to add custom or preselected product

## Response

I have completed the single-screen Shopping List application, focusing on a clean, modern layout:

Clean Slate & Emerald Theme: Styled with a minimal color palette and generous spacing to keep the focus on readability, utilizing an AnimatePresence layout for smooth interactions when adding or removing items.

Preselected & Custom Inputs: Added a convenient item insertion panel containing both a text field for custom entries and a horizontal quick-add list of suggested products. Suggested products automatically grey out once added to prevent duplicates.

Editing & Persistence: List items support complete inline editing, completion toggles, and deletion. All list changes automatically persist to your browser's local storage so you won't lose your list if you refresh the page.
